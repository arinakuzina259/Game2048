# 2048 Game

Классическая игра 2048, реализованная на Java с использованием Swing.

## Возможности

- Полная игровая механика 2048
- Поле 4x4
- Сдвиг и объединение плиток (клавиши стрелок)
- Подсчёт очков
- Сохранение лучшего рекорда
- Адаптивный интерфейс (масштабирование при изменении размера окна)
- Кроссплатформенность (Windows, Linux, macOS)

## Требования

- Java JDK 8 или выше

## Запуск

### Из командной строки

```bash
javac com/minesweeper/*.java
java com.minesweeper.Game2048Main
